# KrishiConnect Web

React PWA frontend for KrishiConnect - Social platform for Indian farmers.

## Setup

```bash
npm install
cp .env.example .env
# Edit .env with API URL
npm run dev
```

## Features

- Auth (Register with OTP, Login)
- Feed with posts (create, like, comment)
- User profiles
- Chat (conversation list)
- Q&A browse
- Market prices
- Weather

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Query
- Zustand
- React Hook Form + Zod
